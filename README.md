# 🌌 Dynamic Scientific Calculator

A sleek, responsive, and fully functional scientific calculator built with **Vanilla JavaScript**, **HTML5**, and **CSS3**. This project features a modern "Glassmorphism" design with a dynamic animated background.

## 🚀 Live Demo
Check out the live application here: 
[https://scientificcalculator260.netlify.app/](https://scientificcalculator260.netlify.app/)

---

## ✨ Features
* **Scientific Functions:** Supports Trigonometry (sin, cos, tan), Logarithms, Square Roots, and Powers.
* **Constants:** Quick access to $\pi$ and $e$.
* **Advanced Logic:** Handles the order of operations (PEMDAS) automatically.
* **Dynamic UI:** * Deep blue and purple animated mesh background.
    * Interactive button scaling and glow effects.
    * Responsive design for both desktop and mobile.

## 🛠️ Technologies Used
* **HTML5:** Semantic structure.
* **CSS3:** Custom animations, Flexbox/Grid, and linear gradients.
* **JavaScript:** Logic handling and mathematical computations using the `Math` object.
* **Netlify:** Continuous deployment and hosting.
